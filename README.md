## Requirements

Before diving in, make sure you have the following prerequisites installed on your system:

### 1. External Tools
- **Python (v3.8 or higher)**: You need Python installed in your machine. You can download and install it from [Python's Official Website](https://www.python.org/downloads/). 
- **Git**: To clone down the project locally. You can download it directly from [Git's website](https://git-scm.com/downloads).

### 2. Python Packages
- `CodeVideoRenderer`: This handles the heavy lifting regarding animating the code and letting the 'camera' track the syntax lines.

All Python dependencies are bundled in the `requirements.txt` file setup.

---

## Step-by-Step Setup Instructions

Here is how you can set up this project locally on your machine. Follow the steps sequentially:

### Step 1: Clone the Repository
Open down your terminal natively and clone the repository onto your system and change into its directory.

```bash
git clone https://github.com/JPB17387/Your-Repository-Name.git
cd Your-Repository-Name
```
*(Make sure to change `Your-Repository-Name` with the actual repository name.)*

### Step 2: Create a Virtual Environment (Highly Recommended)
Creating an isolated environment keeps all project dependencies organized, safe, and confined to this workspace. Do not skip this!

```bash
python -m venv .venv
```

Now activate your virtual environment:

- **Windows:**
  ```bash
  .venv\Scripts\activate
  ```
- **macOS / Linux:**
  ```bash
  source .venv/bin/activate
  ```

*You can verify you are in an active environment if you see `(.venv)` displayed on the left of your command prompt window.*

### Step 3: Install Required Project Dependencies
Now, install all necessary internal program tools by using `pip` from the requirements file.
```bash
pip install -r requirements.txt
```

### Step 4: Run the Visualizer
In your main configuration, the script has been set up to parse a provided script (e.g., `javascript_script.js` or `my_script.py`) into visual content. 

Start the rendering sequence by running:
```bash
python main.py
```
*Depending on the length of your code this may take a few seconds. Once finished, a unique time-stamped video file generated by the CodeVideoRenderer under-the-hood engine defaults out.*

Enjoy animating your code!
